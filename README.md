## 一个学习编程的经典误区

世界不只有Web，还有很多问题需要使用编程来解决。不要把思维局限在Web上，这只是编程的一个应用方向。

## python 能做什么

- 爬虫
- 大数据与数据分析（Spark）
- 自动化运维与自动化测试
- web开发：Flask、Django
- 机器学习：Tensor Flow
- 胶水语言：混合其他如C++、Java等来编程。能够把用其他语言制作的各种模块（尤其是C/C++）很轻松地联结在一起
